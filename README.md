# OCR File Cleaner

クリップボードにコピーされた画像を解析し、対応するローカルの動画ファイルを自動的に検出・削除するPythonツールです。

YouTubeやクラウドストレージにアップロードした動画の一覧を画像として与え、元動画のファイルを効率よく削除するために開発しました。

OCR特有の誤読を各環境に合わせて補正するアルゴリズムを搭載しています。

## 構成ファイル
プラットフォームの環境に合わせて2種類のスクリプトを用意しています。

- `delete_from_clipboard_youtube.py`
  - 対象: YouTube Studio の「コンテンツ」一覧画面
  - 特徴: Youtube上にアップロードされた動画タイトルからファイル名を推測し、消去します。

- `delete_from_clipboard_terabox.py`
  - 対象: Terabox のファイル一覧画面
  - 特徴: 表示されているファイル名から拡張子（.mp4）を除去して元のファイル名を推測し、消去します。

## 機能
- クリップボード画像の直接解析 (Win+Shift+S)
- OCRノイズ（カンマ、ピリオド、汚れ）の正規表現による除去
- 対象のファイル名には含まれない「O (オー)」を「0 (ゼロ)」に自動変換する文字置換
- 「3」と「8」の誤読パターンを総当たりで検証するファジーマッチング
- 桁数チェックによる誤削除防止

## 前提条件・環境
本プログラムの動作には、Tesseract OCR のインストールが必要です。

コード内では以下の環境に合わせてパスを設定しています。
- Tesseract OCR (インストール先: C:\Program Files\Tesseract-OCR\tesseract.exe)
- 削除対象ディレクトリ: J:\
